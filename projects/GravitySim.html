GravitySim


NOTE: this project is still a work in progress, and I am actively adding new features and improvements. I will try and keep this page updated with the latest features and improvements, but for the most up-to-date information, please refer to the repository linked below.


GravitySim is a physics simulation program I created using C++ and OpenGL. It simulates the gravitational interactions between celestial bodies, allowing users to visualize the dynamics of planetary systems and galaxies.

The program includes features such as adjustable parameters for mass, velocity, and time step, as well as real-time rendering of the simulation. This project is a significant learning experience in both programming and physics, and it has been a rewarding way to explore the complexities of gravitational forces.

Controls


  • WASD Keys: Pan the view up, left, down, and right respectively.
  • Left Click: Add a new body to the simulation at the cursor location. Also can interact with menu items


While in Free Place mode, you can adjust the properties of placing a new body using in the bottom menu.

While in Analyze mode, you can click on existing bodies to view their properties and trajectories.

You can pause time and step through the simulation about one frame at a time.
